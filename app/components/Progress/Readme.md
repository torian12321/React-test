Basic
```
<div>
	<Progress />
	<Progress val={80} />
	<Progress val={20} />
</div>
```

States
```
initialState = {val: 50};

<div>
	<Range 
		value    = {state.val}
		onChange = {(e) => setState({val: e})}
		texted
	/>

	<Progress val={state.val} state='main'/>
	<Progress val={state.val} state='sec'/>
	<Progress val={state.val} state='success'/>
	<Progress val={state.val} state='danger'/>
	<Progress val={state.val} state='warning'/>
</div>
```
