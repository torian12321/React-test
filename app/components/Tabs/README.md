Usage
---------------

```
<Tabs selected={1}>
    <Tabs.Panel label="Tab 1">
        This is my tab 1 contents!!!!
    </Tabs.Panel>
    <Tabs.Panel label="Tab 2">
        <div>This is my tab 2 contents!</div>
    </Tabs.Panel>
    <Tabs.Panel label="Tab 3">
        <h1>Content 3</h1>
        <Loader/>
    </Tabs.Panel>
</Tabs>
```
