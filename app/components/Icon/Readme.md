Add SVG icons form [ReactIcons](https://www.npmjs.com/package/react-icons) or similar fonts, so there is *no need to add external .css or .js* libraries.
You can even add your own SVG files.


Default:
```
<Icon />
```

Avaliable Icons list:
```
<div>
	<Icon name='twitter' />
	<Icon name='facebook' />
	<Icon name='cog' />
</div>
```
